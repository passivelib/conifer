About conifer

Conifer Electronics provides Electrical Design Automation (EDA) tools tailored for integrated circuit design, with a particular focus on high-frequency applications.

Conifer’s mission is to enable RF integrated circuit designers to quickly synthesize optimal RF passive networks, including matching networks, using real component models.

our products
RF TOOL CHAIN

Make Cell
(mcell):

Generates high-quality, DRC-clean layouts for a wide range of custom passive components, including inductors, transformers, baluns, and T-coils.

Learn more

Make Modell
(mmod):

Fits electromagnetic simulation results to a SPICE model (for a single geometry) or to a parametrized SPICE model (for multiple geometries), enabling integration into circuit simulations.
Fits electromagnetic simulation results to a custom proprietary model (M-MODEL) which can be used by mrf, enabling synthesis of passive RF circuits (such as matching networks).

Learn more

Make RF Network
(mrf):

Utilizes M-MODELs generated by mmod to identify the optimal matching network for a given set of design parameters.

Learn more
DOCUMENTATION TOOLS

MStudio
(mstd):

Creates publication quality circuit diagrams in vector graphics.

Learn more

Make Documentation
(mdoc):

Consolidates simulation results produced by Cadence Virtuoso into a single, easy-to- manage HTML file that can be opened in any web browser without requiring security modifications, and that can be embedded into a pdf or word document, facilitating documentation and sharing of results and ensuring a smoother and safer user experience.
